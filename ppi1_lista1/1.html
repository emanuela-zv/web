Diferença entre Web e Internet

 A web foi criada como uma forma de compartilhar informações e funciona sobre a internet, e usa o protocolo cliente-servidor.
Enquanto a internet é uma rede de computadores, a principal rede de comunicação global.
Ou seja, para a web funcionar é necessário que haja internet, enquanto o contrário não é verdadeiro, pois anterior à web é a internet.
