O que é o HTTP

O protocolo HTTP (Hypertext Transfer Protocol - Protocolo de Tranferência de Hipertexto) permite a conversa entre duas máquinas, a máquina cliente e a máquina servidor. Assim, ele é o protocolo de transferências de informações WWW.

Ele permite que o cliente faça uma requisição e obtenha uma resposta.

Além disso, a interação entre o cliente e o servidor, ocorre através do protocolo TCP (Transmission Control Protocol) o que auxilia na garantia de que ela seja bem executada.

Um exemplo comum, é realizar uma pesquisa no Google (que seria um cliente). A partir disso, ele trará uma resposta do servidor, conforme a pesquisa.
